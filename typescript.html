TypeScript Basics, What Is It?

TypeScript is an open-source language which builds on JavaScript, one of the world’s most used tools, by adding static type definitions. Types provide a way to describe the shape of an object, providing better documentation, and allowing TypeScript to validate that your code is working correctly.

  • TypeScript is a lanuage that adds value to JavaScript by constructing your code in a manner that is always being checked for errors
  • A JavaScript superset, meaning it’s a layer around JS with more methods and that makes you follow a certain way of development that you don’t have to otherwise in vanilla (like having to set the types of your variables).
  • Both the browser and node js cant excecute Typescript
  • Typescript is a tool, a compiler that you run over your javascript code to create Typescript. Typescript will provide a easier way to do something while than is compiled to more complex Javascript code
  • Typescript adds types to javascript which is a feature.
