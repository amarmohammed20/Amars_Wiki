CSS Basics - Getting started

  • CSS stands for cascading style sheets - core technology to designing a website. The presentation of the website
  • CSS resource Webplatofrm.org
  • The three meathods you can apply CSS to a website
    • Inline style - Write the CSS in the HTML file inside the tag. Not good practice. Inline style will override all other style applied
    • Internal styles - Placed in the HTML documents in the head tag and placed in style tags. A inline style will override this. The HTML file becomes to big and the css is realoded after every refresh
    • External CSS - This is an external .css file which is managed seperatley. Need to link the file to the HTML - link tag with rel= which is relationship which is stylesheet and href
  • You can also import a style sheet - This is placed on the HTML in the head tags inbetween style tags. Example @import "";
    You can also and is preferred to place the import in the .css stylesheet which is seperate.

CSS - Basic Selectors

  • A css selector is created of two parts, the property and the value
  • A selector lets us target HTML elements.
  • A universal selector selects every element on the page at once, overriding all other css. * {}
  • A type selector lets you select a element type on the page, also called html selectors. Example p {}. Wrtie these in the order they appear in the HTML file to keep things neat
  • ID selector, this is were you place a ID in the HTML tag and than call is out in the external CSS file using #. Example #primary-content {}
    An element can only have one ID and only one element can have the same ID name
  • ID's can also be used as anchor elements for the page. You can call the ID as an anchor through a href
  • Class selectors - A class is like a ID but an ID is used for one element and a class is used for many elements
    To call a class selector you use . Example .primary-content . Multiple elements can share the same class.
    An example of were this might be used is we could have multiple divs that require the same formatting. We use a class once instead of multiple ID's
  • You can add multiple classes in a HTML element which means that in the css file you can have one class that done the common things that most elements need, like font.
  • ID's carry more weight than classes as they point to a particular element.
  • You can use descendant selectors in CSS to capture the decendants of a element you are applying style to. An example for the span selector which is a child of the header

    descendant selectors can be used with classes and ID's
  • Pseudo-Classes are similiar to classes but specifies a special state of the selected element(s).
    For example, :hover can be used to change a button's color when the user's pointer hovers over it.


    Pseudo-Classes can be applied to any HTML elements
  • To comment in CSS you use /* */ - Shortcut is control /
