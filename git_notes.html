The below is what I have learned on the first day

    • Once added you can commit "git commit -m "comment" which places the file in the commited area
  • you can assign a name and email address to your user on git providing more information about you, useful to when other look at who made changes in the aduit of git

    • git config -l - Enables you to check the username and email and other git information, to exit use "q".
