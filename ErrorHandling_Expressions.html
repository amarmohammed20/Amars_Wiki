Error Handling, Try Catch Block

Errors Will Happen! When executing JavaScript code, different errors can occur. Errors can be coding errors made by the programmer, errors due to wrong input, and other unforeseeable things.

  • The try statement lets you test a block of code for errors. This could be a basic function or more complex code
  • The catch statement lets you handle the error. For catch you pass in a parameter of your choice and call on that paramter and its properties. Properties like message, name, instanceof TypeError and many more
  • try and catch work toghther and can handle errors without stopping the script

  • Finally can be added at the end and this ensures that the block of code in finally runs
  • The throw statement lets you create custom errors. You could say something like if something is not equal to something throw error which could be custom.

Regular Expressions

A regular expression is a sequence of characters that forms a search pattern. The search pattern can be used for text search and text replace operations.
